<h1>Pinwheel Infrastructure (AWS)</h1>

This repository contains **Terraform infrastructure-as-code (IaC)** for provisioning, configuring, and managing **Amazon Web Services (AWS) infrastructure**.

[![Terraform Version](https://img.shields.io/badge/terraform-4.50.0-844FBA?logo=terraform&logoColor=white)](https://www.terraform.io)
</div>

## Purpose

- Provision AWS infrastructure using Terraform
- Support multiple environments (dev / staging / prod)
- Integrate with CI/CD pipelines (TBD)

## Prerequisites

Ensure the following tools are installed and configured:

- Terraform `4.50.0`
- AWS CLI (configured with proper credentials)
- An active AWS account
- Proper IAM permissions (AdministratorAccess or required resource-specific permissions)

## Technology Stack

- Terraform
- AWS
